### Debugging Exercises – Data Science & Computer Vision Admission
This repository contains solutions to a series of debugging exercises required for the admission procedure of the Master’s program in Data Science & Computer Vision.

### Getting Started
To run the exercises locally, follow these steps:
1. **Clone the repository**
   ```bash
   git clone https://github.com/MaryamMirbagheri/NHL_assignment.git

2. **Create an Anaconda environment**

  ```bash
  conda create --name debugging_exercises
  conda activate debugging_exercises
  ```

4. **Insall core packages**
* 
  ```bash
  conda install numpy matplotlib

5. **Jupyter**
  ```bash
  pip install notebook
  ```
  
6. **Insall PyTorch**
  ```bash
  pip install torch torchvision
  ```
  
7. **Run Jupyter Notebook**
 ```bash
 jupyter notebook
 ```
 
8. **Install Additional Dependencies (if needed)**
If any package is missing while running the notebook, install it using:
 ```bash
 pip install <package-name>
 ```
 
9. **Deactivate the Environment**
 ```bash
 conda deactivate
